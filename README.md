## Features
- Conversational AI powered by Google Gemini 1.5 Flash
- Persistent conversation memory across sessions
- Voice output (pyttsx3 – offline TTS)
- Voice input (SpeechRecognition + microphone)
- Coloured terminal UI with typing animation
- Open websites / run system commands via ACTION responses
- Built-in commands: help, history, clear, config, voice on/off, mic, time, exit

## Built-in Commands
| Command    | Description                    |
|------------|--------------------------------|
| help       | Show all commands              |
| history    | Show conversation history      |
| clear      | Clear memory                   |
| config     | Show configuration             |
| voice on   | Enable TTS voice output        |
| voice off  | Disable TTS                    |
| mic        | Speak one message via mic      |
| time       | Show current date/time         |
| exit/quit  | Exit JARVIS
